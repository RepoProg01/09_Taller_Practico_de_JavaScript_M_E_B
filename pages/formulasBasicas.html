AREAS Y PERIMETROS

Area del Triangulo :

Formula = Base * Altura / 2

Perimetro del Triangulo :

Formula = Lado1 + Lado2 + Base

Area del Cuadrado :

Formula = Lado * Lado

Perimetro del Cuadrado :

Formula = 4 * Lado

Area del Rectangulo :

Formula = Base * Altura

Perimetro del Rectangulo :

Formula = (2 * Base) + (2 * Lado)

Area del Paralelogramo :

Formula = base * altura

Perimetro del Paralelogramo :

Formula = 2 x (b + L)

Area del Trapecio :

Formula = ((B + b) * h) / 2

Perimetro del Trapecio :

Formula = B + b + L1 + L2

Area del Rombo :

Formula = D x d / 2

Perimetro del Rombo :

Formula = L1 + L2 + L3 + L4

Area del pentagono :

Formula = perimetro x apotema / 2

Perimetro del Pentagono :

Formula = 5 x L

Area del Hexagono :

Formula = (perimetro x apotema) / 2

Perimetro del Hexagono :

Formula = 6 * Lado

Area del Circulo :

Formula = pi x r2

Circunferncia :

Formula = pi * diametro
